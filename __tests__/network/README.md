# Tests Réseau Archivés

Les tests unitaires pour les fonctionnalités réseau (NetworkStats, network.service, etc.) ont été archivés dans le cadre de la restructuration de l'application.

Ces tests concernaient les fonctionnalités MCP (Network, Channel, etc.) qui ont été déplacées dans le dossier `archive/` de l'application.

Pour référence, les tests archivés peuvent être consultés dans le fichier `archive/network-tests.tsx`.
